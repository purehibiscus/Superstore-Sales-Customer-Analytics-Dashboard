**Overview**


This project analyzes retail transaction data to uncover insights into sales performance, customer behavior, subscriptions, discounts, and product demand using Excel / Power BI.


**Business Questions Answered**

1. Which products and categories drive the most revenue?
2. How does seasonality impact sales?
3. What payment and shipping methods perform best?
4. How valuable are subscribers compared to non-subscribers?
5. Which products are high-value vs high-volume?


**KPIs**


1. Total Sales Revenue
2. Total Orders
3. Average Order Value (AOV)
4. Repeat Purchase Rate
5. Subscriber Revenue %
6. Discount Usage %
7. Express Shipping Adoption %


**Dashboards**

**Page 1 – Sales Overview**

1. Seasonal trends
2. Geographic sales map
3. Category and payment analysis


**Page 2 – Customer & Product Insights**

1. Gender behavior
2. Purchase frequency vs value
3. Shipping performance
4. Size distribution
5. Top products


**Tools & Skills**

1. Excel
2. PivotTables & PivotCharts
3. Advanced Formulas(MEDIAN, FILTER, VLOOKUP, XLOOKUP, IFS, UNIQUE)
4. KPI Design
5. Dashboard UX Best Practices
